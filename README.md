# Streamlit To-Do List Manager

A simple yet powerful task management application built with Streamlit.

## Features
- ✅ Add new tasks
- 🚀 Mark tasks as complete/incomplete
- 🗑️ Delete tasks
- 💾 Save/Load tasks from JSON file
- 📊 Task statistics
- 🎨 Clean and intuitive UI
